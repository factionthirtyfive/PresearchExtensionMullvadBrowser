# PresearchExtensionMullvadBrowser
Presearch Extension For MullvadBrowser

What does it do?
- You can use Presearch in your Mullvad Browser search bar.

How to use it?
Download and Install Mullvad Browser.
Move "{441ae0e7-c955-4b0e-8209-99462af458db}" into C:\Users\"USERNAME"\AppData\Roaming\Mullvad\MullvadBrowser\Profiles\"Version".default-release\extensions
